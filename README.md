## What I Learned from the Video 
1. Git Basics — What Git Is

Git is a distributed version control system that helps you track changes to your code over time, collaborate with others, and safely experiment without losing work.

2. git add: Stage Your Changes
git add takes changes you made in your working directory and adds them to the staging area (also called the index). The staging area is where you prepare what will go into your next commit.

5. Branching — Work in Parallel
A branch is simply a pointer (label) to a specific commit in the history — like a bookmark you can move around as you add more commits.
The default branch is often called main or master
You can create new branches to work on features without affecting the main codebase
